📋 PROJECT STRUCTURE
====================

You have 4 skeleton files to complete:

FRONTEND (React):
  └─ App.js

BACKEND (Node.js):
  └─ server.js
     Person 2: Claude + Stability AI integration
     Person 4: ElevenLabs + FFmpeg integration

🚀 SETUP STEPS (Do this FIRST before coding)
==============================================

PERSON 1 & 3 (Frontend):
  1. Create React app:
     $ npx create-react-app manifestation
     $ cd manifestation
     $ npm install axios

  2. Replace src/App.js with the App.js skeleton

  3. Start the app:
     $ npm start
     (It will open at localhost:3000)

PERSON 2 & 4 (Backend):
  1. Create backend folder:
     $ mkdir manifestation-backend
     $ cd manifestation-backend

  2. Initialize:
     $ npm init -y
     $ npm install express axios cors dotenv fluent-ffmpeg

  3. Create files:
     - Copy server.js here
     - Copy .env.template as .env

  4. Get API Keys (CRITICAL):
     - Claude: https://console.anthropic.com/
     - Stability AI: https://platform.stability.ai/account/api-keys
     - ElevenLabs: https://elevenlabs.io/

  5. Add keys to .env file

  6. Install FFmpeg on your machine:
     Mac:    brew install ffmpeg
     Windows: choco install ffmpeg OR download from ffmpeg.org
     Ubuntu:  sudo apt-get install ffmpeg

  7. Start backend:
     $ node server.js


⚙️ HOW THEY CONNECT
===================

Frontend → Backend Flow:
  1. User types prompt in textarea
  2. Clicks "Create My Video" button
  3. Frontend sends POST to http://localhost:5000/api/generate
  4. Backend receives prompt
  5. Backend calls Claude API
  6. Backend calls Stability AI (parallel for all scenes)
  7. Backend calls ElevenLabs
  8. Backend calls FFmpeg
  9. Backend returns video URL
  10. Frontend displays video

❓ TROUBLESHOOTING
==================

Backend won't start:
  - Check .env file has all 3 API keys
  - Check FFmpeg is installed: ffmpeg -version
  - Check port 5000 is available

No video appears:
  - Check browser console for errors (F12)
  - Check terminal logs for error messages
  - Make sure backend is running
  - Make sure frontend is calling correct URL

API errors:
  - Double-check API keys in .env
  - Make sure APIs have credit/free tier available
